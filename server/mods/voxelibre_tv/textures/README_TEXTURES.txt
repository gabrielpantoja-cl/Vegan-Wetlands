TEXTURAS NECESARIAS PARA voxelibre_tv
========================================

El mod necesita las siguientes texturas (archivos PNG de 16x16 píxeles):

1. voxelibre_tv_top.png
   - Textura para la parte superior/inferior del TV
   - Sugerencia: Color gris oscuro o negro (carcasa del TV)

2. voxelibre_tv_left.png
   - Textura para el lado izquierdo del TV
   - Sugerencia: Similar a top, con detalles de carcasa

3. voxelibre_tv_right.png
   - Textura para el lado derecho del TV
   - Sugerencia: Similar a left

4. voxelibre_tv_back.png
   - Textura para la parte trasera del TV
   - Sugerencia: Gris oscuro con detalles de ventilación

5. voxelibre_tv_off.png
   - Textura para la pantalla apagada
   - Sugerencia: Pantalla negra o gris muy oscuro

6. voxelibre_tv_channel_1.png
   - Textura animada para canal 1 (puede ser 16x32 para 2 frames)
   - Sugerencia: Contenido educativo sobre animales o naturaleza

7. voxelibre_tv_channel_2.png
   - Textura animada para canal 2 (puede ser 16x32 para 2 frames)
   - Sugerencia: Contenido educativo sobre alimentación vegana

8. voxelibre_tv_channel_3.png
   - Textura animada para canal 3 (puede ser 16x32 para 2 frames)
   - Sugerencia: Contenido educativo sobre compasión

CÓMO CREAR TEXTURAS ANIMADAS:
- Las texturas de canales deben ser verticales (16x32 para 2 frames)
- Cada frame de 16x16 se muestra durante 1 segundo (2 segundos total por ciclo)
- El juego alternará entre los frames automáticamente

HERRAMIENTAS SUGERIDAS:
- GIMP (gratuito)
- Pixelorama (editor de pixel art gratuito)
- Aseprite (editor de pixel art de pago)

ALTERNATIVA TEMPORAL:
Puedes copiar texturas de otros mods o usar texturas simples de colores
hasta tener texturas definitivas.
